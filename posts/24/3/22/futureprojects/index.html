Things I'm cancelling/putting on hold

Sinthalia

This is an idea I had for a 2D top down game, that would incorporate both open-world, sandbox and tower defense elements, inspired by games like No Man's Sky and Mindustry, and incorporating various elements of both of them.
A close friend (gozzer) even made a basic concept prototype of the game, complete with sprites and a basic block building system. (Psst - you can actually play this prototype in your browser. There's a secret somewhere on this website that will lead you to it if you're lucky enough to find it!)
Another one of my friends even made some amazing soundtracks for the game, and I have to say, they're pretty damn cool. Unfortunately, as much as I like the idea for this game and the amount of plans I have for it, I've decided to put it on hold for several reasons:

  • Due to certain reasons, I'm probably going to use Godot to make the game if/when I do, so I'll need to start from the beginning.
  • I also need to actually learn Godot properly so I can make the game actually decent-ish.
  • There are some serious flaws with the game idea as it is now, which I have yet to figure out.
  • In general, I don't really have motivation for this project anymore.

Things I'm working on

FORGED (formerly PA-FIS)

FORGED is a storyline I'm working on, and I consider it to be probably my most ambitious project yet. The playable levels are being made in the game Project Arrhythmia, but a substantial part of the story will also take place in cutscene videos or on this website. The storyline is part of a greater PA collection called Project Elevatia by DXL44. I've had a lot of fun making it so far, and have even come up with ways to link into other PE stories. There is a Discord server you can join to discuss the story and other things, however it's currently invite-only. That being said, if you know me on Discord and are actually really interested, please drop me a DM and I'll invite you!
